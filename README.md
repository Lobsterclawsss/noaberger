# Noa Berger - Personal Website

## Stack
- Framework: Next.js 14 (Static Export)
- Styling: Tailwind CSS
- Hosting: Cloudflare Pages
- Backend: Cloudflare Workers + Supabase

## Pages
- / - Hero + Bio
- /work - Projects & case studies
- /writing - Blog posts
- /contact - Contact form

## Deploy
```bash
wrangler pages deploy dist --project-name=noaberger-website
```
